# SwiftUI-5-Tab-Theme-App-Starter
A clean 5 tab SwiftUI starter app with theme switching, profile, game, leaderboard, history, settings, accessibility speech, and app info pages.

Built by Laurand Osmeni

A clean and structured SwiftUI starter app made for GitHub sharing and quick project setup.

## Features

- 5 tab app structure
- Theme switching (Red, Blue, Green, Pink, Orange, Purple)
- Profile tab with nickname, emoji avatar, location, and gender toggle
- Game tab header showing current user info and mute or unmute icon
- Leaderboard tab with searchable dummy users from around the world
- History tab with clear history confirmation
- Settings tab with:
  - Accessibility read out loud toggle
  - App Info
  - Terms of Use
  - Privacy Policy
  - Build version display
- iPhone and iPad friendly layout widths
- Keyboard search and Done support

## Purpose

This project is a reusable SwiftUI app foundation for demos, learning, and starting new apps with a clean tab based structure.
